[機能]
・WARP3Dが出力するbpfファイルから指定した荷重ステップのJ積分値をCSVファイルにダンプします。
・フォーマットはWARP3Dに付属するpacket_readerと同じフォーマット感じです。

[実行環境]
3系のPythonの環境が必要です。環境構築の方法は、基本的にPythonの公式(www.python.org/)から3.x.yのバージョンのインストーラーをダウンロードし、環境を入れるマシンでインストーラーに従って作業を進めるだけです。


[使い方]
WindowsならコマンドプロンプトまたはPowershell上で、Linuxならシェル上で次のコマンドを実行するだけです。

Python dumpDomain.py <入力ファイル(bpf)> <出力ファイル>

ex) Python dumpDomain.py a_wrp.bpf jdomain.csv

出力ファイルに抽出したデータをダンプします。

※"Python"の部分は環境によって適宜読みかえてください。例えば、2系と3系のPythonを両立している場合、Windowsでは"Py -3"となっていたり、Linuxでは"Python3"となっていたりします。

※Pythonコマンドが認識されていない場合はシステム環境変数のパスが通っているかを確認してください。

[注意]
・CSVをExcel等で開くときは","(カンマ)のみを区切り文字として指定してください。

・ダンプされたCSVファイルをExcel等で開くと、J積分値の小数点以下桁数が4桁より小さく表示されるなどで正しく表示されないかもしれませんが、その場合は値が書き込まれているセルの書式設定等で小数点以下の桁数を4桁にすれば正しく表示されます。(WARP3Dに付属するpacket_readerで表示される小数点以下桁数は4桁)
